# Optional extras

Install these only if the user asks for that backend.

## OpenAI / OpenRouter

```bash
uv pip install 'rank-llm[openai]'
```

## vLLM-based OSS listwise reranking

```bash
uv pip install 'rank-llm[vllm]'
```

## Gemini

```bash
uv pip install 'rank-llm[genai]'
```

## Pyserini retriever support

```bash
uv pip install 'rank-llm[pyserini]'
```

## Everything (power users only)

```bash
uv pip install 'rank-llm[all]'
```
